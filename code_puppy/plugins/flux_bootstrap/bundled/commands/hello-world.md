# Hello World in Lesser-Known Languages

## Instructions

Execute the following steps to display "Hello World" in 5 random lesser-known languages:

1. **Select 5 Random Lesser-Known Languages**

   From your knowledge, randomly select 5 lesser-known or underrepresented languages. Prioritize:

   - Languages with fewer than 10 million native speakers
   - Regional or minority languages
   - Indigenous languages
   - Languages with unique scripts or writing systems

   Avoid common languages like English, Spanish, French, German, Chinese, Japanese, Portuguese, Italian, Russian, Arabic, Hindi, and Korean.

2. **Generate the Translations**

   For each selected language, provide the accurate "Hello World" greeting or equivalent phrase in that language. Use the native script when applicable.

3. **Display the Output**

   Present the results in this exact format:

   ```
   🌍 Hello World in 5 Lesser-Known Languages:

   1. [Language]: [Translation]
   2. [Language]: [Translation]
   3. [Language]: [Translation]
   4. [Language]: [Translation]
   5. [Language]: [Translation]
   ```

4. **Add a Brief Context** (Optional)

   After the list, include a one-line fun fact about one of the selected languages.

**Important**: Generate a unique, varied selection on each execution. Aim to surprise the user with interesting and diverse language choices.
